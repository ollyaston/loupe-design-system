# Icons

Icon components and visual indicators that enhance user interfaces with meaningful symbols and visual cues. These components provide consistent iconography across the design system.

## What are Icon Components?

Icon components provide visual symbols and indicators that help users understand interface elements and actions. They offer:

- **Visual clarity** - Clear, recognizable symbols for actions and states
- **Consistency** - Unified icon style across all interfaces
- **Accessibility** - Proper labeling and screen reader support
- **Scalability** - Vector-based icons that work at any size

## Component Categories

### Basic Icons

- **Icon** - Core icon component with multiple variants and sizes
- **IconBadge** - Icons with status badges and indicators
- **Gems** - Specialized icon components for achievements and status

## Usage Guidelines

### When to use Icon Components

- For navigation and menu items
- To indicate actions and states
- For status indicators and notifications
- When space is limited and text isn't practical
- To enhance visual hierarchy and scanning

### Best Practices

- Use icons consistently across similar actions
- Provide text labels for accessibility
- Choose appropriate icon sizes for context
- Ensure icons are recognizable and intuitive
- Test with different screen sizes and resolutions
