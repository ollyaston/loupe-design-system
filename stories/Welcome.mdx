# Welcome to Loupe

**Loupe** is the unified design system—a framework of reusable components, tokens, and patterns.
It's built for speed, clarity, and accessibility, helping designers and engineers craft consistent, scalable, and inclusive experiences.

NOTE: This file documents components for both engineers and LLM code generators. Please keep it up to date when changing components.
